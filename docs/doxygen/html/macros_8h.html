Detailed Description

This file contains the definitions of the macros used in the project.

This file contains the definitions of the macros used in the project. It contains MIN, DEBUG_PRINT, and GEN_ERR and GEN_OK macros (among some others). The DEBUG_PRINT macro is used to print debug messages. It can be used in two ways: DEBUG = 1: DEBUG_PRINT_I, DEBUG_PRINT_W, DEBUG_PRINT_E (all debug messages) DEBUG = 2: DEBUG_PRINT_W, DEBUG_PRINT_E (only warnings and errors)
