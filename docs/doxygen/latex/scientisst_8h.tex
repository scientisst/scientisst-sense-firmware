scientisst-\/firmware main function 

Creates Mutex, initializes nvs, loads op\+\_\+settings from flash, initializes wifi, bluetooth, adc, i2c etc. If pin 1 is enabled, enters config mode. If wifi mode, tries to connect to wifi network. If connection is unsuccessful, enter config mode. Either start web server (config mode), TCP server (wifi mode) or bluetooth. Starts send data task, adc task and adc\+Ext task. If wifi start rcv task; else, start battery task (adc2) task.
